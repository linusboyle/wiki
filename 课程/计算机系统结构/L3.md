# Cache

![](_v_images/20200305113632906_1301722993.png)

局部性：
- 时间局部性：重复访问一个单元（包括循环重复指令）
- 空间局部性：被访问单元的相邻单元被访问（包括顺序执行指令）

Cache设计的问题：
- 映射规则
- 查找算法
- 替换算法
- 写策略

Cache LIne <= Cache Block + 元数据

映射规则：
- 全相联
- 直接映射
- 组相联

## Cache查找

1. 标签比对。Cache目录表项：有效位、标识tag。
2. 顺序查找
3. 并行查找：利用相联存储器（类似Key-Value存储）或者单体多字比较器
